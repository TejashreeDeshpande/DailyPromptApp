# Feature: Sort and Filter

The product team wants to add sort and filter controls to a list screen.

Users should see a list of items that can be sorted by name (A–Z or Z–A) and by a numeric
value (low-to-high or high-to-low). Users should also be able to filter the list by one
or more categories using toggleable chips or checkboxes. The sort and filter options should
apply simultaneously. The list should update immediately whenever the user changes any option,
and reset to the default view when all filters are cleared.
