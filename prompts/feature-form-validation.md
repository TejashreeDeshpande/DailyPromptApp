# Feature: Registration Form

The product team wants to build a user registration form.

The form should include fields for full name, email address, and password. Each field
should be validated in real time as the user types: name must not be empty, email must
follow a valid format, and password must be at least 8 characters. Validation errors
should appear beneath the relevant field. The submit button should remain disabled until
all fields pass validation. On successful submission, a confirmation message should be shown.
