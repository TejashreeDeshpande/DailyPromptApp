# Feature: Stopwatch with Laps

The product team wants to build a stopwatch feature.

Users should be able to start, pause, and reset the stopwatch. The elapsed time should
display in mm:ss.S format and update every 100 milliseconds while running. Users should
be able to record lap times while the stopwatch is running. All recorded laps should be
displayed in a scrollable list below the timer, showing the lap number and the time at
which it was recorded. Resetting the stopwatch should also clear the lap list.
