# Feature: Reusable Selectable List

The product team wants a reusable selectable list component that works for any data type.

Build a generic composable that accepts a list of items, a function to extract a display
label from each item, and a callback for when an item is selected. The component should
highlight the currently selected item. Demonstrate it working for at least two different
data types on the same screen (e.g. a list of countries and a list of categories) without
duplicating the list rendering logic.
