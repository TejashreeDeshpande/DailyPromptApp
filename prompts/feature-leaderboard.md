# Feature: Leaderboard

The product team wants to build a leaderboard feature.

Users should see a ranked list of players sorted by score in descending order. Each row should
display the player's rank, name, and score. If two players have the same score they should share
the same rank and the next rank should be skipped (e.g. 1, 2, 2, 4). Users should be able to
add a new player by entering a name and score, and the list should re-rank automatically.
