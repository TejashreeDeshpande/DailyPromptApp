# Feature: Onboarding Flow

The product team wants to build a first-launch onboarding flow.

New users should be shown a sequence of three screens explaining the app's key features,
with one feature highlighted per screen. Users should be able to swipe between screens or
use Next and Back buttons. A progress indicator should show which step they are on. The
final screen should include a Get Started button that takes the user to the main app. Once
completed, the onboarding flow should not appear again on subsequent launches.
