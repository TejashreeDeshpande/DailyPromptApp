# Feature: Chat UI

The product team wants to build a basic chat screen UI.

Users should see a scrollable list of messages displayed in a chat-bubble layout, with
sent messages aligned to the right and received messages aligned to the left. Each message
should show the text and a timestamp. A text input and a Send button at the bottom of the
screen should allow the user to type and submit new messages. After sending, the list should
automatically scroll to show the newest message.
