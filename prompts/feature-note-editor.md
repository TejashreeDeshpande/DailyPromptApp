# Feature: Note Editor

The product team wants to build a simple note-taking feature.

Users should see a list of saved notes, each showing a title and a short preview of the
content. Tapping a note should open a detail screen where the full note is displayed.
From the detail screen, users should be able to edit both the title and body of the note
and save their changes. Users should also be able to create a new note from the list screen.
