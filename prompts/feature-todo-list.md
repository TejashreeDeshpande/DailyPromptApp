# Feature: To-Do List

The product team wants to build a simple to-do list feature.

Users should be able to add new tasks by typing in a text field and tapping an Add button.
Each task should appear in a scrollable list below the input. Users should be able to mark
a task as complete by tapping it, which visually distinguishes it from incomplete tasks.
Completed tasks should remain in the list but appear differently styled.
